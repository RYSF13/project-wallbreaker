## 📖 项目介绍 | Introduction

**破壁计划 (Project Wallbreaker)** 是一个致力于互联网自由访问的公益项目。
我们在 GitHub 上维护一份实时更新的 VPN/代理 节点库，帮助受网络封锁限制的用户重获访问世界信息的权利。

本项目主要提供 **V2Ray/Trojan/SS** 通用订阅，支持 V2RayN(PC)、v2rayNG(Android)、Shadowrocket(iOS) 等主流客户端。

## 🛠 使用教程 | Usage Guide

### 💻 Windows (使用 V2RayN)

1. 下载并解压 [V2RayN](https://github.com/2dust/v2rayN/releases)。
2. 打开软件，点击菜单栏的 `订阅分组` -> `订阅设置`。
3. 点击 `添加`，在 **地址(url)** 栏填入上方的 **主线路** 链接，备注填写 `破壁计划`。
4. 保存后，返回主界面，点击菜单栏 `订阅分组` -> `更新订阅`。
5. 选择一个节点，右键 `设为活动服务器`，系统代理选择 `自动配置系统代理`。

### 🤖 Android (使用 v2rayNG)

1. 下载安装 [v2rayNG](https://github.com/2dust/v2rayNG/releases)。
2. 点击左上角菜单 -> `订阅设置` -> `+` 号。
3. 填入 **主线路** 链接，备注随意，保存。
4. 返回主界面，点击右上角 `...` -> `更新订阅`。
5. 选中节点，点击右下角 `V` 图标连接。

### 🍎 iOS (使用 Shadowrocket / 小火箭)

1. 打开 Shadowrocket。
2. 点击右上角 `+` 号。
3. 类型选择 `Subscribe` (订阅)。
4. `URL` 栏粘贴 **主线路** 链接，备注填写 `破壁计划`，点击完成。
5. 列表会自动更新，开启连接即可。
